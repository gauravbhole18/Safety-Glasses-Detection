# Safety-Glasses-Detection
# 👓 Eye Glasses Detection using Webcam

A simple computer vision project that detects whether a person is wearing glasses using a webcam. The system performs real-time detection and optionally draws bounding boxes around detected glasses.

## Features
- Real-time webcam detection
- Glasses presence classification
- Bounding box detection for glasses
- FPS display for performance monitoring

Requirements

Python 3.x

OpenCV

Matplotlib

📷 Real-Time Webcam Detection

Run the webcam detection script:

python webcam.py

The script:

Captures webcam frames

Classifies glasses presence

Runs detection if glasses are present

Draws bounding boxes

Displays FPS
